Install / Usage Guide

Install from GitHub for shared usage across projects and teammates.

1. Add gem to host app

group :development do
  gem "rails_agent_annotator", git: "https://github.com/rustyameadows/Rails-Agent-Annotator.git", branch: "main"
end

2. Install generator

bundle install
bin/rails generate rails_agent_annotator:install

3. Run app and annotate

bin/rails server
  1. Enable select mode in overlay.
  2. Click target element and write notes.
  3. Copy markdown and paste into coding agent.
  4. Open Debug from overlay to inspect stored notes.

4. Config (initializer)

RailsAgentAnnotator.configure do |config|
  config.enabled_environments = %w[development]
  config.enabled = Rails.env.development?
  config.mount_path = "/__agent_annotator"
  config.storage_key_prefix = "rails_agent_annotator"
end

v1 persistence is localStorage only (browser-local, per route path).

5. Local development override (optional)

gem "rails_agent_annotator", path: "/absolute/path/to/agent-feedback-gem"

Use this only when iterating on the gem locally.
